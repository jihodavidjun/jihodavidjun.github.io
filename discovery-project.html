Section 1

Discovery Project Idea

Overview

This project explored how to integrate a foundry SRAM macro into a full RTL-to-GDS flow, focusing on real implementation challenges.

Initial Idea / Pitch

I wanted to understand what it takes to make a memory macro work in a real implementation flow beyond RTL.

Why I Chose It

I am interested in physical design, and SRAM integration is a real-world backend challenge.

What I Hoped To Learn

I aimed to learn macro integration, timing validation, and signoff debugging.

Section 2

The Problem

Problem Statement

SRAM macros have fixed layouts, non-standard pins, and complex timing. The challenge was integrating them correctly into the flow.

Why It Mattered

Incorrect macro integration can break a design even if it looks logically correct.
